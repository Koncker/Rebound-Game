This is currently an Unfinished Version of Rebound - A simple 2D game that is like Arkanoid.
Created for the 2D Complete C# Unity Development Course by Ben Tristem and Rick Davidson.

System Requirements:
	Windows 7+ 64 Bit

Installation:
	Installation is not required. Simply download the folder and Double Click on Rebound.exe

Known Bugs:
	- Song does not loop correctly as there is a break.

Change Log:
	- Sound Effects are still missing for some parts
